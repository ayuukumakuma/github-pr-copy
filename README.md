# GitHub PR Copy Tool

WXT-based Chrome extension that adds copy buttons to GitHub pull request pages.

## Requirements

- `mise`
- Node.js 24 and pnpm 10 via `mise install`
- Chrome or Chromium

## Setup

```bash
mise install
vp install
```

## Commands

```bash
vp check
vp test
vp run dev
vp run build
vp run zip
```

## Build Output

- Chrome MV3: `.output/chrome-mv3/`

## Load Unpacked

1. Open `chrome://extensions/`
2. Enable developer mode
3. Choose `Load unpacked`
4. Select `.output/chrome-mv3/`
